## How to use

### Declare service
Add the service as you normally do with bim-gdpr
```javascript
    import { FacebookPixelService } from "../bim-gdpr/services/facebook_pixel/facebook_pixel"

    const facebook_pixel = new FacebookPixelService()
    bGDPR
        .addService(facebook_pixel)
```
